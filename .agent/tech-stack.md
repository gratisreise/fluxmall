### 1단계: Legacy & Essentials (기본기 다지기)

- **Language:** Java 11 (LTS)
- **Framework:** Spring Framework 5.x (Non-Boot)
- **View:** JSP (Java Server Pages), JSTL, Tiles (Layout 설정)
- **Data Access:** JdbcTemplate, Apache Commons DBCP (Connection Pool)
- **Database:** MySQL 5.7 / 8.0
- **Server:** Apache Tomcat 9 (외장 WAS 환경)
- **Build Tool:** Maven (pom.xml 기반 라이브러리 관리)
- **Library:** Log4j2, Jackson (JSON 파싱), Servlet API
- **Structure:** Single Module, Layered Architecture (`.war` 배포)
